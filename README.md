# apple-supply-chain-dashboard
End-to-end supply chain analytics project analyzing Apple supplier performance, supplier risk, operational efficiency, lead time, defect rate, and sustainability metrics using Python and Power BI dashboards.
Apple Supply Chain Performance & Supplier Risk Analysis

This project analyzes Apple’s supply chain ecosystem using a 10,000-row supplier dataset to uncover insights related to supplier performance, operational efficiency, risk management, and sustainability.

The project combines Python (EDA) and Power BI (Dashboarding) to build an end-to-end analytical solution for supply chain decision-making.

**Objectives**
Analyze supplier performance across countries and product categories
Identify high-risk suppliers and risk distribution
Evaluate operational efficiency using lead time and defect rates
Measure sustainability performance across suppliers

**Key Insights**
Malaysia recorded the highest supply volume in the dataset
High-risk supplier records represented the largest risk segment
Display manufacturing showed the highest lead time
iPad and Apple Watch categories had the highest defect rates
India showed strong sustainability performance

This project demonstrates practical business intelligence and data analytics skills in supply chain optimization, risk management, and executive reporting.
